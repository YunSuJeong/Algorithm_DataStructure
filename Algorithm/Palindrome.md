# Palindrome(회문)

## 팰린드롬, 너 누군데?
> 나? 앞에서부터 읽어도 뒤에서부터 읽어도 똑같은 단어, 문장, 숫자, 문자열

ex) dad, 1001, abcdcba, level ...

## 판별 방법
1. 주어진 문자의 **처음과 끝을 비교**
2. 안쪽으로 **한 글자씩 좁혀 가운데까지 비교**했을 때, 서로 같으면 팰린드롬이다.
